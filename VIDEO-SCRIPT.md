## Production Notes

### For Canva:
1. Create a new project — "Instagram Post (Square)" or "Instagram Reel" depending on format
2. Add 5 slides, set durations to: 4s, 6s, 5s, 8s, 7s
3. Upload all 4 images from the Ad Images folder
4. Use "Fade" or "Dissolve" transitions between slides (0.3s)
5. Add text boxes on each slide per the script above
6. For the zoom effect: use Canva's "Pan and Zoom" animation on each image
7. Font recommendation: Inter or Montserrat, bold weight for headlines
8. Export as MP4

### For iMovie:
1. Import all 4 images
2. Set each clip to the durations above (adjust in clip inspector)
3. Apply Ken Burns (zoom) effect on each image as described
4. Add titles using the "Standard" or "Lower Third" title styles
5. Use "Cross Dissolve" transitions (0.3s)
6. For Shot 5 (closing), create a black background in Preview.app or Canva and import it
7. Export at 1080p

### Text Styling:
- **Headlines:** White, bold, 48-64pt, with a subtle drop shadow for readability over images
- **Secondary text:** White or light gray, regular weight, 32-40pt
- **Price ($39):** Blue (#2563eb), bold, 56pt
- **URL:** White, regular weight, 24pt
- Use a semi-transparent dark bar behind text when it overlaps busy parts of an image

### Aspect Ratios:
- **Instagram/Facebook Feed:** 1080x1080 (square)
- **Instagram Reels / TikTok:** 1080x1920 (9:16 vertical)
- **Facebook Video Ad:** 1080x1080 or 1200x628 (landscape)
- The square format works across the most platforms — start there
