# Influencer Outreach

Plan + copy-paste kit for promoting the $39 youth/amateur baseball stats app through
small, niche baseball creators. Built for the current manual-payment reality: PayPal.me
link + manual `isPaid` toggle (no automated checkout or promo codes).

---

## Strategy

For a $39 niche app, **micro + niche beats a big name**. The buyer is a travel-ball/Little
League coach or a baseball parent. Target creators whose entire audience is exactly those
people, even at 15k–80k followers — engagement and buyer-intent matter far more than reach.

Don't bet on one creator. Run a small **test across 2–3 micro creators**, attribute each
with a unique code, then re-invest only in the winner.

### Realistic budget
- Micro coaches (15k–40k): ~$50–250 per post — **start here**
- Mid-tier coaches (100k–200k): ~$300–800 — only after a code converts
- First test total: **$150–400**

## The offer

Lead with a small flat fee + an affiliate code. Low risk to you, motivating for them.

- **Flat fee:** $75–150 for one feed post or Reel + one story
- **Discount code:** e.g. `MASSEY10` → $10 off the $39 app ($29)
- **Affiliate kicker:** $10 per sale their code generates
- **Free account** so they can show it on screen

---

## How attribution works (no code change needed)

Payment is a PayPal.me link and you flip `isPaid` by hand — use that manual step as the
tracker:

1. Give each creator a unique discounted link with the price baked in:
   `paypal.com/paypalme/carlrandrews/29`
2. Tell followers: **"Put code `MASSEY10` in the PayPal note."**
3. When you verify the payment to flip `isPaid`, read the note → you know which creator
   drove the sale. Tally per code = affiliate attribution.

### Traffic attribution (now built in)
Each creator also gets a unique tracking link to the site:
`https://www.baseballstatstracker.com/pricing?ref=massey`

The `?ref=` value is stored in a cookie and attached to that visitor's **SIGNUP** event.
In `/admin` → Analytics → Recent Activity, signups show an amber `ref: massey` badge, so
you can see which creator drives signups even when a buyer forgets the PayPal note.

Use one slug per creator, matched to their code: `?ref=massey` ↔ `MASSEY10`,
`?ref=heat` ↔ `HEAT10`, `?ref=tbs` ↔ `TBS10`.

## Outreach copy

### Version A — short DM (Instagram)

> Hey [Name] — love what you're doing with [specific thing, e.g. "the infield drill
> breakdowns"]. I built **BaseballStatsTracker**, an app for youth/travel coaches and
> parents to track full game stats (batting, pitching, spray charts, pitch-by-pitch).
> Yours is exactly the audience it's for.
>
> Would you be open to a paid shoutout? I'm thinking a small flat fee + your own discount
> code for followers + a free account to try it. Happy to work around what you usually
> charge. Cool if I send details?

> - A free account so you can show it in action
>
> If you're interested, tell me your rate and I'll send a free login and a few talking
> points. Thanks for considering it!
>
> Carl
> BaseballStatsTracker.com

### Talking points to hand them
- "Track every at-bat — no more paper scorebooks"
- "Spray charts + pitch-by-pitch, the kind of stuff travel coaches actually want"
- "Works live during the game, syncs across phones in the dugout"
- "One-time $39, not another monthly subscription"

### Two tips
1. **Personalize the first line every time** — reference an actual recent post. One
   specific detail roughly doubles reply rates vs. generic copy-paste.
2. **Stagger codes + ref slugs** (`MASSEY10`/`?ref=massey`, etc.) so you know exactly
   which creator drives sales before spending more.
